# Visual Identity System — Yohann Doillon

**Brand adjectives:** Direct. Precise. Grounded.

---

## Color Palette

### Primary Background — Deep Navy
| Token | Hex | Usage |
|-------|-----|-------|
| `neutral-950` | `#0A1220` | Page background |
| `neutral-900` | `#121B28` | Dark surface, cards |
| `neutral-800` | `#1F2533` | Elevated surface, code blocks |
| `neutral-700` | `#363330` | Borders, dividers |

### Text — Off-White
| Token | Hex | Usage |
|-------|-----|-------|
| `neutral-50` | `#F4F1EC` | Primary headings |
| `neutral-100` | `#E8E2DA` | Body text |
| `neutral-200` | `#D0C9BE` | Secondary text |
| `neutral-400` | `#8D8476` | Captions, metadata |

### Accent — Electric Teal
| Token | Hex | Usage |
|-------|-----|-------|
| `brand-400` | `#0DD4C2` | Primary links, CTAs (on dark bg) |
| `brand-500` | `#00B8A6` | Interactive states |
| `brand-300` | `#3DE5D5` | Hover states, highlights |
| `brand-600` | `#009486` | Borders, subtle accents |
| `brand-700` | `#007265` | Dark accent surfaces |

### Color Ratios
- Dark backgrounds (navy): 80% of surface area
- Off-white text: primary reading surface
- Teal accent: sparingly — links, CTAs, and 1–2 accent elements per screen
- No gradients. No glows. Flat color only.

## Typography

### Typefaces
| Role | Family | Weight | Notes |
|------|--------|--------|-------|
| Headlines | Inter | 600 (SemiBold) | Geometric sans-serif. Clean, no-frills authority. |
| Body | Inter | 400 (Regular) | Same family. Consistent, readable. |
| Code / Data | JetBrains Mono | 400 | Technical precision. Used for code, metrics, and data callouts. |
| Captions | Inter | 400, italic | Metadata, timestamps, attribution. |

### Type Scale
| Level | Size | Weight | Line Height |
|-------|------|--------|-------------|
| Display | 3rem (48px) | 600 | 1.15 |
| H1 | 2rem (32px) | 600 | 1.2 |
| H2 | 1.5rem (24px) | 600 | 1.3 |
| H3 | 1.25rem (20px) | 600 | 1.35 |
| Body | 1rem (16px) | 400 | 1.75 |
| Small | 0.875rem (14px) | 400 | 1.6 |
| Caption | 0.75rem (12px) | 400 | 1.5 |

### Typography Rules
- Headings: always `neutral-50` on dark backgrounds
- Body copy: `neutral-100` to `neutral-200`
- Links: `brand-400` (#0DD4C2), underline offset 3px
- Max line length: 65ch for body text
- Letter-spacing: -0.01em for headings, 0 for body
- No decorative fonts. No display typefaces.

---

## Spacing & Layout

- Base unit: 4px (0.25rem)
- Standard spacing: 8 / 16 / 24 / 32 / 48 / 64 / 96px
- Max content width: 1200px
- Reading column max: 65ch (~720px)
- Section padding (mobile): 2rem vertical
- Section padding (desktop): 4–6rem vertical

---

## Components & UI Patterns

### Buttons
- **Primary:** `brand-500` bg + `neutral-950` text. No border-radius >8px. No shadows.
- **Secondary:** Transparent bg + `brand-400` border + `brand-400` text.
- **Ghost:** `neutral-700` bg + `neutral-200` text. For low-emphasis actions.

### Cards
- Background: `neutral-900`
- Border: `neutral-700` (1px)
- Border-radius: 12px
- Padding: 1.5rem
- Hover: border shifts to `brand-600`

### Code Blocks
- Background: `neutral-800`
- Border: `neutral-700`
- Font: JetBrains Mono, 14px
- Text: `neutral-200`
- Accent tokens: `brand-300`

### Dividers / Lines
- Use `neutral-700` (1px solid)
- No decorative rules or ornaments

---

## Logo Treatment

- Wordmark only: "Yohann Doillon" in Inter SemiBold
- Tracking: -0.02em
- Color: `neutral-50` on dark; `neutral-950` on light
- Optional: a minimal geometric mark (square or grid of 4 dots, 2x2) — never decorative
- No gradients, no shadows, no glows on the logo

---

## Photography Direction

- Natural light, minimal background clutter
- Color grade: slightly cool, not warm/orange
- Direct eye contact, slight lean-forward posture
- Not stiff, not casual — engaged professional
- No stock photo aesthetic

---

## What This Brand Is Not

- Not startup neon / gradient purple
- Not cold corporate gray
- Not decorative / ornamental
- Not performatively casual (no emojis in copy, no exclamation points)
- Not busy — every element earns its place
